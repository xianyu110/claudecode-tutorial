  ### **Q1：改错了怎么办？**


  別慌，Claude Code 有「時光機」功能。

  **方法一：按 ESC 兩次**

  **方法三：直接說**

  「剛才那個改動我不喜歡，還原回去」

  ### **Q2：我完全不懂代码，怎么知道 Claude 做的对不对？**


  答案：**你不需要懂程式碼，你只需要測試功能**。

  就像你不需要懂汽車發動機，也能知道車能不能開。

  - 新增任務試試，能加上嗎？
  - 標記完成試試，有劃線嗎？
  - 刪除試試，真的刪掉了嗎？
  - 重新整理頁面，資料還在嗎？

  **功能對了，程式碼就是對的。**

  有問題？直接告訴 Claude：「刪除按鈕點了沒反應」，它會自己檢查和修復。

  ### **Q3：为什么有些教程说要装 Node.js？**


  這是個好問題，說明你真的在學習。

  **簡單回答**：如果你用的是我推薦的安裝方式（curl 或 PowerShell 一鍵安裝），就不需要 Node.js。

  **完整解釋**：Claude Code 有多種安裝方式。如果用 NPM 方式（`npm install -g @anthropic-ai/claude-code`），需要先裝 Node.js。但一鍵安裝方式（curl/PowerShell）已經把所有依賴打包好了，直接用就行。

  **對新手來說**：跟著我這篇教程的步驟走，別去碰 NPM 安裝，就不會遇到 Node.js 的問題。

  ### **Q4：我应该从这个开始学编程吗？**

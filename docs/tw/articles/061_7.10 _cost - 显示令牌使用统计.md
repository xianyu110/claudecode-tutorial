# 7.10 /cost - 顯示令牌使用統計

## 概述
`/cost` 命令用於顯示令牌使用統計資訊，幫助你跟蹤和最佳化 Claude Code 的使用成本。

```
## 基本语法

    bash


    /cost

## 功能说明

```
執行 `/cost` 命令後，Claude Code 會：

  1. 顯示當前會話的令牌使用統計
  2. 顯示輸入和輸出令牌數量
  3. 估算使用成本
  4. 顯示使用限制和速率限制狀態

```
## 使用场景

### 1\. 监控使用情况

    bash


    # 查看当前会话的使用统计
    /cost

    # 了解令牌消耗情况

### 2\. 成本跟踪

    bash


    # 定期检查使用成本
    /cost

    # 估算月度使用成本

### 3\. 优化使用

    bash


    # 查看使用统计，优化对话策略
    /cost

    # 根据统计结果调整使用方式

## 令牌类型

### 输入令牌 (Input Tokens)

```
  * 你傳送給 Claude 的文字
  * 包括對話歷史、檔案內容、系統提示等
  * 通常比輸出令牌便宜

### 輸出令牌 (Output Tokens)
  * Claude 生成的文字
  * 包括回答、程式碼、建議等
  * 通常比輸入令牌貴

```

## 使用限制

```
不同的訂閱計劃有不同的使用限制：

### 免費計劃
  * 每日：200,000 令牌
  * 每月：1,000,000 令牌

### 付費計劃
  * 每日：500,000 令牌
  * 每月：5,000,000 令牌

## 注意事項
  1. 令牌統計是估算值，實際計費可能略有差異
  2. 成本基於當前模型的定價
  3. 使用限制會在每日/每月重置
  4. 超出限制後，可能需要升級計劃

```
## 与其他命令的配合使用

    bash


    # 查看使用统计，然后优化
    /cost
    /compact
    /cost

    # 执行流程：
    # 1. 查看当前使用统计
    # 2. 压缩对话历史
    # 3. 再次查看，确认优化效果

## 最佳实践

```
  1. **定期檢查** ：定期使用 `/cost` 檢查使用情況
  2. **及時最佳化** ：當使用率超過 70% 時，考慮最佳化
  3. **合理使用** ：根據任務複雜度選擇合適的模型
  4. **壓縮歷史** ：定期使用 `/compact` 減少令牌消耗

## 總結
`/cost` 命令是 Claude Code 中用於跟蹤使用成本的重要工具。透過使用此命令，你可以：

  * 檢視令牌使用統計
  * 估算使用成本
  * 監控使用限制
  * 最佳化使用策略

合理使用 `/cost` 命令可以幫助你更好地控制使用成本，提高使用效率。

# 7.12 /exit - 退出REPL

## 概述
`/exit` 命令用於退出 Claude Code 的互動式 REPL（Read-Eval-Print Loop）環境，結束當前會話。

```
## 基本语法

    bash


    /exit

## 功能说明

```
執行 `/exit` 命令後，Claude Code 會：

  1. 儲存當前會話狀態（如果配置了自動儲存）
  2. 清理臨時檔案
  3. 關閉所有後臺任務
  4. 退出 REPL 環境
  5. 返回命令列

```
## 使用场景

### 1\. 完成工作后退出

    bash


    # 完成所有工作后
    /exit

    # 退出 Claude Code

### 2\. 需要重启时退出

    bash


    # 需要重启 Claude Code 时
    /exit

    # 然后重新启动

### 3\. 切换项目时退出

    bash


    # 切换到另一个项目前
    /exit

    # 在新项目目录中重新启动

## 注意事项

```
  1. `/exit` 會立即退出，不會提示儲存
  2. 未儲存的對話歷史會丟失
  3. 後臺任務會被終止
  4. 配置和設定會保留

```bash
## 与其他命令的配合使用

    bash


    # 退出前清理
    /export session.md
    /exit

    # 执行流程：
    # 1. 导出当前对话到文件
    # 2. 退出 Claude Code

## 最佳实践

```
  1. **匯出對話** ：退出前使用 `/export` 儲存重要對話
  2. **清理後臺** ：退出前使用 `/bashes` 檢查後臺任務
  3. **儲存狀態** ：確保重要工作已儲存
  4. **正常退出** ：使用 `/exit` 而不是強制終止

## 總結
`/exit` 命令是 Claude Code 中用於退出 REPL 的命令。透過使用此命令，你可以：

  * 正常退出 Claude Code
  * 清理臨時檔案
  * 關閉後臺任務
  * 儲存會話狀態

合理使用 `/exit` 命令可以幫助你安全地結束 Claude Code 會話。

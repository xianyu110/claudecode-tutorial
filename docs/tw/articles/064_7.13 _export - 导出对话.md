# 7.13 /export - 匯出對話

## 概述
`/export` 命令用於將當前對話匯出到檔案或剪貼簿，方便儲存和分享重要的對話內容。

```bash
## 基本语法

    bash


    /export [filename]

## 参数说明

```
引數| 描述
---|---
`[filename]`| 可選，匯出的檔名。如果不提供，則匯出到剪貼簿

## 功能說明
執行 `/export` 命令後，Claude Code 會：

  1. 收集當前會話的完整對話歷史
  2. 格式化為 Markdown 格式
  3. 匯出到指定檔案或剪貼簿
  4. 顯示匯出成功確認

```bash
## 使用场景

### 1\. 导出到文件

    bash


    # 导出对话到文件
    /export session-2024-01-01.md

    # 对话会保存到当前目录

### 2\. 导出到剪贴板

    bash


    # 导出到剪贴板
    /export

    # 对话内容会复制到剪贴板

### 3\. 导出重要对话

    bash


    # 导出重要的对话记录
    /export important-discussion.md

    # 保存以便后续参考

### 4\. 分享对话

    bash


    # 导出对话以便分享
    /export share-with-team.md

    # 可以通过邮件或其他方式分享

## 导出格式

```
匯出的檔案採用 Markdown 格式，包含：

```
    markdown


    # Claude Code Session

    Date: 2024-01-01 14:30:00
    Model: claude-3-5-sonnet-20241022

    ## Conversation

    ### User
    你好，请帮我写一个 Python 函数。

    ### Assistant
    好的，这是一个示例函数...

    ### User
    谢谢，能解释一下吗？

    ### Assistant
    当然，这个函数的作用是...

    ---
    End of Session

## 注意事项

```
  1. 匯出的檔案會覆蓋同名檔案
  2. 匯出到剪貼簿會替換剪貼簿內容
  3. 匯出包含完整的對話歷史
  4. 敏感資訊會被包含在匯出中

```bash
## 与其他命令的配合使用

    bash


    # 导出对话，然后退出
    /export session.md
    /exit

    # 执行流程：
    # 1. 导出当前对话到文件
    # 2. 退出 Claude Code

## 最佳实践

```
  1. **定期匯出** ：定期匯出重要對話，避免丟失
  2. **命名規範** ：使用有意義的檔名，便於管理
  3. **敏感資訊** ：匯出前檢查是否包含敏感資訊
  4. **版本控制** ：可以將匯出的檔案加入版本控制

## 總結
`/export` 命令是 Claude Code 中用於儲存對話的重要工具。透過使用此命令，你可以：

  * 匯出對話到檔案
  * 匯出對話到剪貼簿
  * 儲存重要對話記錄
  * 分享對話內容

合理使用 `/export` 命令可以幫助你儲存和管理重要的對話內容。

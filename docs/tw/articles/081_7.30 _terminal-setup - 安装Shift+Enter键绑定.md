# 7.30 /terminal-setup - 安裝Shift+Enter鍵繫結

## 概述
`/terminal-setup` 命令用於安裝 Shift+Enter 鍵繫結，以便在終端中輸入多行文字。此功能僅限 iTerm2 和 VSCode 終端。

```
## 基本语法

    bash


    /terminal-setup

## 功能说明

```
執行 `/terminal-setup` 命令後，Claude Code 會：

  1. 檢測當前終端型別
  2. 安裝 Shift+Enter 鍵繫結
  3. 配置多行輸入模式
  4. 顯示安裝成功確認

```
## 支持的终端

### iTerm2

```
  * **支援** : ✓ 完全支援
  * **配置** : 自動配置
  * **功能** : Shift+Enter 換行，Enter 傳送

### VSCode 終端
  * **支援** : ✓ 完全支援
  * **配置** : 自動配置
  * **功能** : Shift+Enter 換行，Enter 傳送

### 其他終端
  * **支援** : ✗ 不支援
  * **建議** : 使用 iTerm2 或 VSCode 終端

```javascript
## 使用场景

### 1\. 在 iTerm2 中使用

    bash


    # 在 iTerm2 中安装键绑定
    /terminal-setup

    # 配置完成后，可以使用 Shift+Enter 换行

### 2\. 在 VSCode 终端中使用

    bash


    # 在 VSCode 终端中安装键绑定
    /terminal-setup

    # 配置完成后，可以使用 Shift+Enter 换行

### 3\. 输入多行文本

    bash


    # 安装后，可以输入多行文本
    function hello() {
      console.log("Hello, World!");
    }

    # 使用 Shift+Enter 换行，Enter 发送

## 键绑定说明

### 默认行为

```
  * **Enter** : 傳送命令
  * **Shift+Enter** : 換行

### 安裝後行為
  * **Enter** : 傳送命令
  * **Shift+Enter** : 換行（不傳送）

## 注意事項
  1. 僅支援 iTerm2 和 VSCode 終端
  2. 需要重啟終端才能生效
  3. 鍵繫結會影響所有終端會話
  4. 可以手動解除安裝鍵繫結

```
## 卸载键绑定

### iTerm2

    bash


    # 在 iTerm2 中手动卸载
    1. 打开 Preferences
    2. 选择 Profiles -> Keys
    3. 删除 Shift+Enter 键绑定

### VSCode

    bash


    # 在 VSCode 中手动卸载
    1. 打开 Settings
    2. 搜索 "terminal.integrated.sendShiftEnter"
    3. 禁用该选项

## 与其他命令的配合使用

    bash


    # 安装键绑定，然后使用
    /terminal-setup
    # 输入多行文本

## 最佳实践

```
  1. **確認終端** ：確認使用支援的終端
  2. **重啟終端** ：安裝後重啟終端
  3. **測試功能** ：安裝後測試鍵繫結
  4. **備份配置** ：解除安裝前備份配置

## 總結
`/terminal-setup` 命令是 Claude Code 中用於配置終端鍵繫結的命令。透過使用此命令，你可以：

  * 安裝 Shift+Enter 鍵繫結
  * 在終端中輸入多行文字
  * 提高終端使用效率
  * 改善多行輸入體驗

合理使用 `/terminal-setup` 命令可以幫助你在終端中更方便地輸入多行文字。

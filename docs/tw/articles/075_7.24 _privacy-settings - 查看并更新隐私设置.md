# 7.24 /privacy-settings - 檢視並更新隱私設定

## 概述
`/privacy-settings` 命令用於檢視並更新 Claude Code 的隱私設定，控制資料使用和隱私保護。

```
## 基本语法

    bash


    /privacy-settings

## 功能说明

```
執行 `/privacy-settings` 命令後，Claude Code 會：

  1. 開啟隱私設定介面
  2. 顯示當前隱私配置
  3. 允許修改隱私選項
  4. 儲存隱私設定

```
## 隐私设置选项

### 数据使用

```
  * **對話資料** : 是否允許使用對話資料改進服務
  * **程式碼資料** : 是否允許使用程式碼資料改進服務
  * **匿名化** : 是否對資料進行匿名化處理

### 資料保留
  * **對話歷史** : 對話歷史的保留時間
  * **錯誤日誌** : 錯誤日誌的保留時間
  * **使用統計** : 使用統計的保留時間

### 資料共享
  * **第三方共享** : 是否與第三方共享資料
  * **研究用途** : 是否用於研究目的
  * **匿名共享** : 是否允許匿名共享資料

```

    # 查看当前隐私设置
    /privacy-settings

    # 显示所有隐私配置

### 2\. 更新隐私设置

    bash


    # 更新隐私设置
    /privacy-settings

    # 修改隐私选项

### 3\. 保护敏感数据

    bash


    # 配置隐私设置以保护敏感数据
    /privacy-settings

    # 禁用数据共享

## 隐私设置示例

    json


    {
      "dataUsage": {
        "conversationData": false,
        "codeData": false,
        "anonymization": true
      },
      "dataRetention": {
        "conversationHistory": "30days",
        "errorLogs": "7days",
        "usageStatistics": "90days"

      "dataSharing": {
        "thirdParty": false,
        "research": false,
        "anonymousSharing": false
      }
    }

## 注意事项

```
  1. 隱私設定會影響資料使用政策
  2. 某些功能可能需要特定的隱私設定
  3. 隱私設定儲存在配置檔案中
  4. 更改設定會影響當前和未來的會話


## 總結
`/privacy-settings` 命令是 Claude Code 中用於隱私管理的重要命令。透過使用此命令，你可以：

  * 檢視當前隱私設定
  * 更新隱私配置
  * 控制資料使用和共享
  * 保護敏感資訊

合理使用 `/privacy-settings` 命令可以幫助你更好地保護隱私和資料安全。

# 7.19 /memory - 編輯CLAUDE.md記憶體檔案

## 概述

`/memory` 命令用於編輯 `CLAUDE.md` 記憶體檔案，該檔案包含專案的上下文資訊和重要知識。

## 基本语法

    bash


    /memory

## 功能说明


執行 `/memory` 命令後，Claude Code 會：

  1. 開啟 `CLAUDE.md` 檔案
  2. 在編輯器中顯示檔案內容
  3. 允許你編輯檔案
  4. 儲存更改後，更新專案上下文

## CLAUDE.md 檔案

`CLAUDE.md` 檔案通常包含：

  1. **專案概述** ：專案的基本資訊和目標
  2. **架構說明** ：專案架構和技術棧
  3. **程式碼規範** ：編碼標準和最佳實踐
  4. **重要說明** ：需要注意的事項
  5. **上下文資訊** ：AI 需要了解的專案細節

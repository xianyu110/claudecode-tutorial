# 11.3 熱門 MCP 伺服器

MCP 生態系統中有許多流行的伺服器，可以擴充套件 Claude Code 的功能，讓 AI 助手能夠與各種工具和服務無縫整合。

## 开发工具

### GitHub


**功能** : 程式碼託管和協作平臺整合

**安装** :

    bash


    claude mcp add --transport http github https://api.githubcopilot.com/mcp/

**用途** :


  * 建立和管理 Pull Requests
  * 檢視和建立 Issues
  * 程式碼搜尋和審查
  * 倉庫管理
  * 團隊協作

**示例** :

    bash


    审查 PR #456
    创建新问题：修复登录 bug
    查看我的所有打开的 PR
    搜索仓库中包含 "authentication" 的代码

### GitLab


**功能** : 程式碼託管和 CI/CD 平臺整合

**安装** :

    bash


    claude mcp add --transport http gitlab https://gitlab.com/api/mcp

**用途** :


  * 管理 Merge Requests
  * CI/CD 管道管理
  * 專案管理
  * 程式碼審查
  * 自動化部署

### VS Code

## 数据库

### PostgreSQL


**功能** : PostgreSQL 資料庫查詢和操作

**安装** :

    bash


    claude mcp add --transport stdio postgres -- npx -y @bytebase/dbhub \
    --dsn "postgresql://user:pass@localhost:5432/db"

**用途** :


  * 查詢資料庫
  * 分析資料
  * 生成報告
  * 資料視覺化
  * 資料庫管理

**示例** :

    bash


    查询本月总收入
    显示订单表结构
    查找 90 天未购买的用户
    分析用户购买行为

### MySQL


**功能** : MySQL 資料庫查詢和操作

**安装** :

**用途** :


  * 查詢 MySQL 資料庫
  * 資料分析
  * 報表生成
  * 資料匯入匯出
  * 資料庫維護

### MongoDB

## 云服务

### AWS


**功能** : AWS 雲服務整合

**安装** :

    bash


    claude mcp add --transport stdio aws -- npx -y @anthropic-aws/mcp-server \

**用途** :

  * CloudWatch 監控

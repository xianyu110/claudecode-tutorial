# 2.2 VS Code外掛安裝

## 2.2.1 安裝VS Code
Claude Code提供了VS Code外掛，因此在安裝Claude Code外掛之前，需要先確保已安裝VS Code。

### 2.2.1.1 檢查VS Code是否已安裝
開啟終端或命令提示符，執行以下命令檢查VS Code是否已安裝：

```
    bash


    # 检查VS Code版本
    code --version

```
如果輸出顯示VS Code版本資訊，則說明VS Code已安裝，可以跳過安裝步驟。

### 2.2.1.2 安裝VS Code
如果沒有安裝VS Code，可以透過以下方式安裝：

#### 方式一：使用官方安裝包
  1. 訪問VS Code官方網站：<https://code.visualstudio.com/>
  2. 下載適合你作業系統的安裝包
  3. 執行安裝程式，按照提示完成安裝

#### 方式二：使用包管理器（推薦）
**對於macOS：**

```
    bash


    # 使用Homebrew安装
    brew install --cask visual-studio-code

```
**對於Linux：**

```python

    # Ubuntu/Debian
```

安裝完VS Code後，現在可以安裝Claude Code外掛了。

### 2.2.2.1 從VS Code市場安裝（推薦）
  1. 開啟VS Code
  2. 點選左側邊欄的**擴充套件** 圖示（或使用快捷鍵`Ctrl+Shift+X`/`Cmd+Shift+X`）
  3. 在搜尋框中輸入`Claude Code`
  4. 在搜尋結果中找到Claude Code外掛，點選**安裝** 按鈕
  5. 安裝完成後，外掛會自動啟用

### 2.2.2.2 手動安裝外掛
如果無法從VS Code市場安裝外掛，可以使用以下方式手動安裝：

  1. 訪問Claude Code外掛的GitHub倉庫或官方網站，下載最新的VSIX檔案
  2. 開啟VS Code
  3. 點選左側邊欄的**擴充套件** 圖示（或使用快捷鍵`Ctrl+Shift+X`/`Cmd+Shift+X`）
  4. 點選擴充套件面板右上角的**更多操作** 按鈕（三個點）
  5. 選擇**從VSIX安裝**
  6. 選擇下載的VSIX檔案，點選**安裝**
  7. 安裝完成後，重新載入VS Code以啟用外掛

### 2.2.2.3 驗證外掛安裝
安裝完成後，可以透過以下方式驗證外掛是否成功安裝：

  1. 開啟VS Code
  2. 點選左側邊欄的**Claude Code** 圖示（如果已新增到邊欄）
  3. 或者，使用快捷鍵`Ctrl+Shift+P`/`Cmd+Shift+P`開啟命令面板
  4. 輸入`Claude Code`，如果看到相關命令，則說明外掛已成功安裝

### 2.2.2.4 外掛更新
Claude Code外掛會定期更新，提供新功能和修復bug。可以透過以下方式更新外掛：

  1. 開啟VS Code
  2. 點選左側邊欄的**擴充套件** 圖示（或使用快捷鍵`Ctrl+Shift+X`/`Cmd+Shift+X`）
  3. 點選擴充套件面板右上角的**更多操作** 按鈕（三個點）
  4. 選擇**檢查更新**
  5. 如果有更新可用，點選**更新** 按鈕即可更新外掛

或者，在擴充套件面板中找到Claude Code外掛，如果有更新可用，會顯示**更新** 按鈕。

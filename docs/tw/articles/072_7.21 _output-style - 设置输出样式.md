# 7.21 /output-style - 設定輸出樣式

## 概述
`/output-style` 命令用於直接設定輸出樣式或從選擇選單中選擇，允許你自定義 Claude Code 的輸出格式。

```
## 基本语法

    bash


    /output-style [style]

## 参数说明

```
引數| 描述
---|---
`[style]`| 可選，指定輸出樣式名稱。如果不提供，則顯示樣式選擇選單

```
## 可用样式

### default

```
  * **描述** : 預設樣式
  * **特點** : 平衡的格式，適合大多數場景
  * **適用場景** : 通用任務

### concise
  * **描述** : 簡潔樣式
  * **特點** : 精簡輸出，去除冗餘資訊
  * **適用場景** : 快速查詢、簡單任務

### detailed
  * **描述** : 詳細樣式
  * **特點** : 提供詳細的解釋和示例
  * **適用場景** : 學習、教學、複雜任務

### code-focused
  * **描述** : 程式碼優先樣式
  * **特點** : 側重程式碼，減少文字說明
  * **適用場景** : 程式碼生成、除錯

### documentation
  * **描述** : 文件樣式
  * **特點** : 格式化為文件，結構清晰
  * **適用場景** : 文件編寫、技術說明

```javascript

    ## 注意事项

## 最佳实践

```
  1. **任務匹配** ：根據任務型別選擇合適的樣式
  2. **靈活切換** ：根據需要隨時切換樣式
  3. **團隊統一** ：團隊內部統一使用相同的樣式
  4. **定期評估** ：定期評估樣式選擇是否合理

## 總結
`/output-style` 命令是 Claude Code 中用於自定義輸出樣式的重要命令。透過使用此命令，你可以：

  * 檢視可用的輸出樣式
  * 選擇最適合當前任務的樣式
  * 自定義輸出格式
  * 最佳化閱讀體驗

合理使用 `/output-style` 命令可以幫助你獲得更符合需求的輸出格式。

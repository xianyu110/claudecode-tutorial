# 7.32 /usage - 顯示使用限制和速率限制狀態

## 概述
`/usage` 命令用於顯示計劃使用限制和速率限制狀態，幫助你瞭解當前使用情況和剩餘配額。

```
## 基本语法

    bash


    /usage

## 功能说明

```
執行 `/usage` 命令後，Claude Code 會：

  1. 顯示當前使用量
  2. 顯示使用限制
  3. 顯示速率限制狀態
  4. 顯示配額剩餘情況

```
## 使用信息

### 令牌使用

```
  * 當前使用量
  * 每日限制
  * 每月限制
  * 使用百分比

### 速率限制
  * 請求速率
  * 剩餘請求數
  * 重置時間
  * 限制狀態

### 計劃資訊
  * 當前計劃型別
  * 計劃限制
  * 升級選項

```yaml

## 计划类型

### 免费计划

```
  * 每日限制: 200,000 令牌
  * 每月限制: 1,000,000 令牌
  * 速率限制: 20 請求/分鐘

### Pro 計劃
  * 每日限制: 500,000 令牌
  * 每月限制: 5,000,000 令牌
  * 速率限制: 50 請求/分鐘

### Enterprise 計劃
  * 每日限制: 2,000,000 令牌
  * 每月限制: 20,000,000 令牌
  * 速率限制: 100 請求/分鐘

## 注意事項
  1. 使用限制會在每日/每月重置
  2. 速率限制按分鐘計算
  3. 超出限制後需要等待或升級計劃
  4. 使用量實時更新

```
## 与其他命令的配合使用

    bash


    # 查看使用情况，然后优化
    /usage
    /compact

    # 执行流程：
    # 1. 查看使用情况
    # 2. 压缩对话以减少令牌使用

## 最佳实践

```
  1. **定期檢查** ：定期檢視使用情況
  2. **提前規劃** ：提前規劃使用量
  3. **最佳化使用** ：最佳化對話以減少令牌消耗
  4. **及時升級** ：接近限制時考慮升級計劃

## 總結
`/usage` 命令是 Claude Code 中用於檢視使用情況的重要命令。透過使用此命令，你可以：

  * 檢視當前使用量
  * 檢視使用限制
  * 檢視速率限制狀態
  * 瞭解剩餘配額

合理使用 `/usage` 命令可以幫助你更好地管理和規劃使用配額。

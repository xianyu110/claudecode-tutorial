# 3.1 介面介紹與導航

## 3.1.1 概述

Claude Code提供了直觀的介面，方便開發者進行程式碼開發和協作。熟悉介面佈局和導航方式是使用Claude Code的基礎，它可以幫助你快速找到所需功能，提高工作效率。本節將詳細介紹Claude Code的介面組成和導航方法。

## 3.1.2 VS Code插件界面

### 3.1.2.1 活动栏


VS Code的活動欄位於左側，包含各種功能圖示。Claude Code的圖示通常位於活動欄的底部或中部，圖示是一個帶有Claude標誌的藍色圖示。

點選Claude Code圖示，會開啟Claude Code的側邊欄，顯示主要功能區域。

### 3.1.2.2 側邊欄

Claude Code的側邊欄包含以下主要區域：

  1. **歡迎區** ：首次使用時顯示歡迎資訊和快速開始指南
  2. **對話區** ：顯示與Claude Code的對話歷史
  3. **檔案區** ：顯示專案的檔案結構
  4. **命令區** ：提供常用命令的快捷訪問
  5. **設定區** ：提供快速設定選項

### 3.1.2.3 編輯器區

編輯器區是VS Code的主要工作區域，用於編輯程式碼檔案。Claude Code會在編輯器區中生成和修改程式碼。

### 3.1.2.4 狀態列

狀態列位於VS Code的底部，顯示當前的編輯狀態、分支資訊、語言模式等。Claude Code會在狀態列中顯示當前的連線狀態和模型資訊。

### 3.1.2.5 面板區

面板區位於編輯器區的底部，用於顯示輸出、終端、除錯控制檯等。Claude Code會在面板區中顯示執行結果和日誌資訊。

## 3.1.3 命令行工具界面

### 3.1.3.1 交互式界面


Claude Code命令列工具提供了互動式介面，類似於終端中的聊天介面。在互動式介面中，可以直接輸入命令和訊息與Claude Code進行互動。

### 3.1.3.2 命令提示符

命令列工具的命令提示符通常顯示為：

claude>

在命令提示符後輸入命令或訊息，按Enter鍵執行。

### 3.1.3.3 輸出格式

命令列工具的輸出格式通常為：

```
    bash


    [Claude] 这是Claude Code的响应内容

## 3.1.4 导航方法

### 3.1.4.1 在VS Code中导航

```

  1. **活動欄導航** ：點選活動欄中的Claude Code圖示，開啟或關閉側邊欄
  2. **側邊欄導航** ：在側邊欄中點選不同的選項卡，切換功能區域
  3. **檔案導航** ：在檔案區中瀏覽和開啟專案檔案
  4. **命令導航** ：使用快捷鍵或命令面板訪問Claude Code命令

### 3.1.4.2 快捷鍵

Claude Code提供了以下常用快捷鍵：

快捷鍵| 功能
---|---

`Ctrl+Shift+C` / `Cmd+Shift+C`| 開啟Claude Code側邊欄
`Ctrl+Shift+P` / `Cmd+Shift+P`| 開啟命令面板，輸入"Claude Code"檢視所有命令
`Ctrl+Enter` / `Cmd+Enter`| 傳送訊息到Claude Code
`Esc`| 關閉當前對話或面板

### 3.1.4.3 命令面板

可以使用VS Code的命令面板訪問Claude Code的所有命令：

  1. 按下`Ctrl+Shift+P` / `Cmd+Shift+P`開啟命令面板
  2. 輸入"Claude Code"，會顯示所有可用的Claude Code命令
  3. 選擇需要的命令，按Enter鍵執行

## 3.1.5 界面定制

### 3.1.5.1 调整侧边栏大小


可以透過拖動側邊欄的邊緣來調整其寬度，以適應你的工作習慣。

### 3.1.5.2 隱藏/顯示功能區

可以在VS Code的設定中隱藏或顯示Claude Code的某些功能區：

  1. 開啟VS Code設定
  2. 搜尋 "Claude Code"
  3. 找到"UI"相關設定
  4. 根據需要啟用或禁用功能區

### 3.1.5.3 自定義主題

Claude Code會自動適應VS Code的主題設定。可以在VS Code的設定中修改主題：

  1. 開啟VS Code設定
  2. 搜尋 "Theme"
  3. 選擇你喜歡的主題

## 3.1.6 常见问题与解决方案

### 3.1.6.1 侧边栏不显示


**問題** ：點選Claude Code圖示後，側邊欄不顯示

**解決方案** ：

  * 檢查VS Code是否已更新到最新版本
  * 檢查Claude Code外掛是否已更新到最新版本
  * 嘗試重新啟動VS Code
  * 嘗試解除安裝並重新安裝Claude Code外掛

### 3.1.6.2 介面顯示異常

**問題** ：Claude Code介面顯示異常，如按鈕不顯示、佈局混亂等

**解決方案** ：

  * 檢查VS Code是否已更新到最新版本
  * 檢查Claude Code外掛是否已更新到最新版本
  * 嘗試清除VS Code的快取
  * 嘗試切換到不同的主題

### 3.1.6.3 快捷鍵衝突

**問題** ：Claude Code的快捷鍵與其他外掛或VS Code內建功能衝突

**解決方案** ：

  * 開啟VS Code的鍵盤快捷方式設定
  * 搜尋並修改衝突的快捷鍵
  * 或在Claude Code設定中修改快捷鍵

## 3.1.7 導航最佳實踐

  1. **熟悉快捷鍵** ：記住常用快捷鍵可以提高導航效率
  2. **自定義佈局** ：根據工作習慣調整介面佈局
  3. **使用命令面板** ：命令面板是訪問所有功能的便捷方式
  4. **定期清理對話** ：定期清理對話歷史，保持介面整潔
  5. **使用快速訪問** ：將常用功能新增到快速訪問區域

## 3.1.8 總結

Claude Code的介面設計直觀易用，透過熟悉介面佈局和導航方法，可以提高使用Claude Code的效率。VS Code外掛提供了豐富的視覺化介面，而命令列工具提供了簡潔的終端介面，開發者可以根據自己的偏好選擇使用方式。

在使用Claude Code時，建議花一些時間熟悉介面佈局和導航方法，這將有助於你更快地找到所需功能，提高工作效率。

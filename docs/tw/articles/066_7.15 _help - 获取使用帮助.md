# 7.15 /help - 獲取使用幫助

## 概述
`/help` 命令用於獲取 Claude Code 的使用幫助，顯示所有可用的命令和功能說明。

```
## 基本语法

    bash


    /help [command]

## 参数说明

```
引數| 描述
---|---
`[command]`| 可選，指定要檢視的命令名稱。如果不提供，則顯示所有命令

```bash
## 功能说明

### 查看所有命令

    bash


    /help

    # 显示所有可用的斜杠命令

### 查看特定命令

    bash


    /help init

    # 显示 /init 命令的详细帮助

## 使用场景

### 1\. 查看所有可用命令

    bash


    # 新手入门，查看所有命令
    /help

    # 了解 Claude Code 的功能

### 2\. 查看特定命令帮助

    bash


    # 忘记某个命令的用法
    /help compact

    # 查看该命令的详细说明

### 3\. 探索新功能

    bash


    # 发现新功能
    /help

    # 查看所有命令，了解新功能

## 注意事项

```
  1. `/help` 命令不會修改任何檔案或設定
  2. 幫助資訊是實時更新的
  3. 自定義命令會顯示在幫助中
  4. 外掛命令也會顯示在幫助中

```
## 与其他命令的配合使用

    bash


    # 查看帮助，然后使用命令
    /help init
    /init

    # 执行流程：
    # 1. 查看 /init 命令的帮助
    # 2. 了解用法后执行命令

## 最佳实践

```
  1. **新手必用** ：首次使用時，先執行 `/help` 瞭解所有命令
  2. **忘記用法** ：忘記某個命令的用法時，使用 `/help <command>` 檢視
  3. **定期檢視** ：定期檢視幫助，瞭解新功能和命令
  4. **探索功能** ：透過 `/help` 發現你可能不知道的功能

## 總結
`/help` 命令是 Claude Code 中獲取幫助的核心命令。透過使用此命令，你可以：

  * 檢視所有可用的命令
  * 瞭解特定命令的用法
  * 探索新功能
  * 快速找到需要的命令

合理使用 `/help` 命令可以幫助你更好地掌握 Claude Code 的功能。

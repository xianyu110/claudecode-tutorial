# 7.20 /model - 選擇或更改AI模型

## 概述

`/model` 命令用於選擇或更改 Claude Code 使用的 AI 模型，允許你根據任務需求選擇最合適的模型。

## 基本语法

    bash


    /model [model-name]

## 参数说明


引數| 描述
---|---

`[model-name]`| 可選，指定要使用的模型名稱。如果不提供，則顯示可用模型列表

## 可用模型

### Claude 3.5 Sonnet


  * **名稱** : `claude-3-5-sonnet-20241022`
  * **特點** : 平衡效能和速度
  * **適用場景** : 通用任務、程式碼生成、文件編寫

### Claude 3.5 Haiku


  * **名稱** : `claude-3-5-haiku-20241022`
  * **特點** : 快速響應，成本低
  * **適用場景** : 簡單任務、快速查詢、成本敏感場景

### Claude 3 Opus


  * **名稱** : `claude-3-opus-20240229`
  * **特點** : 最強效能，成本高
  * **適用場景** : 複雜任務、深度分析、高質量要求

```
## 使用场景

### 1\. 查看可用模型

    bash


    # 查看所有可用的模型
    /model

    # 显示模型列表和特点

### 2\. 选择模型

    bash


    # 选择特定模型
    /model claude-3-5-sonnet-20241022

    # 切换到该模型

### 3\. 根据任务选择

    bash


    # 简单任务使用快速模型
    /model claude-3-5-haiku-20241022

    # 复杂任务使用高性能模型
    /model claude-3-opus-20240229

## 模型选择建议

### 代码生成

    bash


    # 代码生成推荐使用 Sonnet
    /model claude-3-5-sonnet-20241022

### 文档编写

    bash


    # 文档编写推荐使用 Sonnet
    /model claude-3-5-sonnet-20241022

### 简单查询

    bash


    # 简单查询推荐使用 Haiku
    /model claude-3-5-haiku-20241022

### 复杂分析

    bash


    # 复杂分析推荐使用 Opus
    /model claude-3-opus-20240229

## 注意事项

```

  1. 不同模型的成本不同
  2. 模型切換會影響當前會話
  3. 某些功能可能需要特定模型
  4. 模型效能取決於任務複雜度

## 与其他命令的配合使用

    bash


    # 查看模型，然后选择
    /model
    /model claude-3-5-sonnet-20241022

    # 执行流程：
    # 1. 查看可用模型
    # 2. 选择合适的模型

## 最佳实践


  1. **任務匹配** ：根據任務複雜度選擇合適的模型
  2. **成本考慮** ：在簡單任務上使用成本較低的模型
  3. **效能優先** ：在重要任務上使用高效能模型
  4. **定期評估** ：定期評估模型選擇是否合理

## 總結

`/model` 命令是 Claude Code 中用於選擇 AI 模型的重要命令。透過使用此命令，你可以：

  * 檢視可用的 AI 模型
  * 選擇最適合當前任務的模型
  * 平衡效能和成本
  * 最佳化使用體驗

合理使用 `/model` 命令可以幫助你根據任務需求選擇最合適的 AI 模型。

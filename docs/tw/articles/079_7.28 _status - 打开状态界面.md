# 7.28 /status - 開啟狀態介面

## 概述

`/status` 命令用於開啟 Claude Code 的狀態介面（狀態選項卡），顯示版本、模型、賬戶和連線性資訊。

## 基本语法

    bash


    /status

## 功能说明


執行 `/status` 命令後，Claude Code 會：

  1. 開啟狀態介面
  2. 顯示系統資訊
  3. 顯示模型資訊
  4. 顯示賬戶資訊
  5. 顯示連線狀態

## 状态信息

### 系统信息


  * Claude Code 版本
  * 作業系統
  * Node.js 版本
  * 記憶體使用

### 模型資訊


  * 當前使用的模型
  * 模型版本
  * 模型狀態

### 賬戶資訊


  * 登入狀態
  * 賬戶型別
  * 使用限制

### 連線狀態


  * API 連線狀態
  * 網路延遲
  * 最後連線時間

```
## 使用场景

### 1\. 查看系统状态

    bash


    # 查看系统状态
    /status

    # 显示所有状态信息

### 2\. 检查连接

    bash


    # 检查 API 连接状态
    /status

    # 查看连接信息

### 3\. 查看使用情况

    bash


    # 查看使用情况
    /status

    # 显示使用限制和当前使用量

## 注意事项

```

  1. 狀態資訊實時更新
  2. 某些資訊需要網路連線
  3. 狀態介面不會自動重新整理
  4. 可以手動重新整理狀態

## 与其他命令的配合使用

    bash


    # 查看状态，然后检查健康
    /status
    /doctor

    # 执行流程：
    # 1. 查看状态信息
    # 2. 运行健康检查

## 最佳实践


  1. **定期檢查** ：定期檢視狀態，瞭解系統情況
  2. **問題診斷** ：遇到問題時，先檢視狀態
  3. **連線監控** ：監控 API 連線狀態
  4. **使用跟蹤** ：跟蹤使用情況，避免超限

## 總結

`/status` 命令是 Claude Code 中用於檢視狀態的重要命令。透過使用此命令，你可以：

  * 檢視系統資訊
  * 檢視模型資訊
  * 檢視賬戶資訊
  * 檢視連線狀態

合理使用 `/status` 命令可以幫助你瞭解 Claude Code 的執行狀態。

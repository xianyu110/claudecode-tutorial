# 7.31 /todos - 列出當前待辦事項

## 概述

`/todos` 命令用於列出當前待辦事項，幫助你跟蹤和管理任務進度。

## 基本语法

    bash


    /todos [subcommand]

## 子命令


子命令| 描述
---|---

無引數| 列出所有待辦事項
`add`| 新增新的待辦事項
`complete`| 標記待辦事項為完成
`remove`| 刪除待辦事項
`clear`| 清除所有待辦事項

```
## 功能说明

### 列出待办事项

    bash


    /todos

    # 显示所有待办事项及其状态

### 添加待办事项

    bash


    /todos add

    # 按照提示输入待办事项

### 完成待办事项

    bash


    /todos complete 1

    # 标记待办事项 #1 为完成

### 删除待办事项

    bash


    /todos remove 2

    # 删除待办事项 #2

## 使用场景

### 1\. 查看待办事项

    bash


    # 查看所有待办事项
    /todos

    # 了解当前任务列表

### 2\. 添加任务

    bash


    # 添加新的待办事项
    /todos add

    # 输入任务描述

### 3\. 完成任务

    bash


    # 标记任务为完成
    /todos complete 1

    # 更新任务状态

### 4\. 清理任务

    bash


    # 清除已完成的任务
    /todos clear

    # 清空待办事项列表

## 待办事项示例

    bash


    待办事项列表

    待完成: 4 个

## 注意事项

```

  1. 待辦事項儲存在本地
  2. 待辦事項不會自動同步
  3. 可以手動編輯待辦事項檔案
  4. 清除操作無法撤銷

## 与其他命令的配合使用

    bash


    # 查看待办事项，然后完成任务
    /todos
    /todos complete 1

    # 执行流程：
    # 1. 查看待办事项列表
    # 2. 完成指定任务

## 最佳实践


  1. **定期更新** ：定期更新待辦事項列表
  2. **明確任務** ：使用清晰的任務描述
  3. **優先順序排序** ：按優先順序排序任務
  4. **及時完成** ：及時完成任務並更新狀態

## 總結

`/todos` 命令是 Claude Code 中用於任務管理的重要命令。透過使用此命令，你可以：

  * 檢視待辦事項列表
  * 新增新的任務
  * 標記任務為完成
  * 刪除任務

合理使用 `/todos` 命令可以幫助你更好地跟蹤和管理任務進度。

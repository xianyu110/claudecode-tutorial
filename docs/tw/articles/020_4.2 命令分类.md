# 4.2 命令分類

Claude Code 的命令按照功能和用途可以分為不同的類別。理解這些分類有助於您快速找到需要的命令，並更好地理解命令系統的整體結構。

```
## 主要命令分类

### 1\. CLI 命令

```
CLI 命令是直接在命令列中執行的命令，用於啟動 Claude Code、執行一次性任務等。主要包括：

  * `claude`：啟動互動式 REPL
  * `claude "query"`：使用初始提示啟動 REPL
  * `claude -p "query"`：透過 SDK 查詢，然後退出
  * `cat file | claude -p "query"`：處理管道內容
  * `claude -c`：繼續最近的對話
  * `claude -r "<session-id>" "query"`：按 ID 恢復會話

### 2\. CLI 標誌
CLI 標誌是與 CLI 命令一起使用的選項，用於自定義 Claude Code 的行為。主要包括：

  * `--print`, `-p`：列印響應而不進入互動模式
  * `--verbose`：啟用詳細日誌記錄
  * `--model`：設定會話模型
  * `--continue`：載入最近的對話
  * `--resume`：恢復特定會話

### 3\. 斜槓命令
斜槓命令是在互動式會話中使用的命令，以 `/` 開頭。主要包括：

  * `/init`：初始化命令
  * `/plugin`：外掛管理命令
  * `/config`：配置 Claude Code
  * `/clear`：清除命令歷史
  * `/agents`：管理子代理

## 按功能分類
除了按型別分類，命令還可以按功能分類：

### 會話管理類
  * 啟動會話：`claude`
  * 繼續會話：`claude -c`、`claude --continue`
  * 恢復會話：`claude -r`、`claude --resume`
  * 清除歷史：`/clear`

### 配置管理類
  * 系統配置：`/config`
  * 模型配置：`--model`

### 外掛管理類
  * 外掛管理：`/plugin`

### 工具和代理類
  * 子代理管理：`/agents`

## 命令分類的意義
理解命令分類可以幫助您：

  1. **快速定位** ：根據需要的功能快速找到對應的命令
  2. **系統學習** ：按照分類系統地學習命令，避免遺漏
  3. **高效使用** ：根據不同的使用場景選擇最合適的命令型別
  4. **擴充套件開發** ：瞭解命令分類有助於開發外掛和自定義命令

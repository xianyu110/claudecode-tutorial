# 7.17 /login - 切換Anthropic賬戶

## 概述
`/login` 命令用於切換 Anthropic 賬戶，允許你在不同的 Anthropic 賬戶之間切換。

```
## 基本语法

    bash


    /login

## 功能说明

```
執行 `/login` 命令後，Claude Code 會：

  1. 開啟瀏覽器進行身份驗證
  2. 提示你登入 Anthropic 賬戶
  3. 獲取訪問令牌
  4. 儲存令牌到本地配置
  5. 切換到新賬戶

```
## 使用场景

### 1\. 切换账户

    bash


    # 切换到另一个 Anthropic 账户
    /login

    # 按照提示完成身份验证

### 2\. 重新认证

    bash


    # 当令牌过期时
    /login

    # 重新获取访问令牌

### 3\. 首次登录

    bash


    # 首次使用 Claude Code
    /login

    # 登录 Anthropic 账户

## 登录流程

```
  1. 執行 `/login` 命令
  2. 瀏覽器自動開啟 Anthropic 登入頁面
  3. 輸入賬戶憑據
  4. 授權 Claude Code 訪問
  5. 返回命令列，登入成功

## 注意事項
  1. 需要有效的 Anthropic 賬戶
  2. 需要網際網路連線
  3. 令牌會儲存在本地配置檔案中
  4. 切換賬戶會影響當前會話
  5. 舊賬戶的令牌會被替換

```
## 与其他命令的配合使用

    bash


    # 登录后检查状态
    /login
    /status

    # 执行流程：
    # 1. 登录 Anthropic 账户
    # 2. 查看账户状态

## 最佳实践

```
  1. **賬戶安全** ：不要在公共裝置上儲存令牌
  2. **定期更新** ：定期重新登入以更新令牌
  3. **賬戶管理** ：為不同用途使用不同的賬戶
  4. **退出登入** ：在公共裝置上使用後，記得 `/logout`

## 總結
`/login` 命令是 Claude Code 中用於賬戶管理的重要命令。透過使用此命令，你可以：

  * 切換 Anthropic 賬戶
  * 重新獲取訪問令牌
  * 管理多個賬戶
  * 保持賬戶安全

合理使用 `/login` 命令可以幫助你靈活地管理 Anthropic 賬戶。

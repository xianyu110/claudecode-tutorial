# 7.18 /logout - 從Anthropic賬戶登出

## 概述

`/logout` 命令用於從 Anthropic 賬戶登出，清除本地儲存的訪問令牌。

## 基本语法

    bash


    /logout

## 功能说明


執行 `/logout` 命令後，Claude Code 會：

  1. 清除本地儲存的訪問令牌
  2. 斷開與 Anthropic API 的連線
  3. 顯示登出成功確認
  4. 需要重新登入才能使用

```
## 使用场景

### 1\. 在公共设备上登出

    bash


    # 在公共设备上使用后
    /logout

    # 清除访问令牌，保护账户安全

### 2\. 切换账户前登出

    bash


    # 切换到另一个账户前
    /logout
    /login

    # 先登出，再登录新账户

### 3\. 令牌问题

    bash


    # 当令牌出现问题时
    /logout
    /login

    # 重新登录获取新令牌

## 注意事项

```

  1. 登出後無法使用 Claude Code
  2. 需要重新登入才能繼續使用
  3. 本地儲存的配置不會丟失
  4. 訪問令牌會被完全清除
  5. 會話狀態會被重置

## 与其他命令的配合使用

    bash


    # 登出后重新登录
    /logout
    /login

    # 执行流程：
    # 1. 登出当前账户
    # 2. 登录新账户或重新登录

## 最佳实践


  1. **公共裝置** ：在公共裝置上使用後，務必登出
  2. **賬戶切換** ：切換賬戶前先登出
  3. **定期登出** ：定期登出並重新登入，更新令牌
  4. **安全第一** ：不要在共享裝置上儲存令牌

## 總結

`/logout` 命令是 Claude Code 中用於賬戶安全的重要命令。透過使用此命令，你可以：

  * 從 Anthropic 賬戶登出
  * 清除本地訪問令牌
  * 保護賬戶安全
  * 為切換賬戶做準備

合理使用 `/logout` 命令可以幫助你保護 Anthropic 賬戶的安全。

# 7.23 /pr_comments - 檢視拉取請求評論

## 概述
`/pr_comments` 命令用於檢視拉取請求（Pull Request）的評論，幫助你瞭解程式碼審查的反饋。

```
## 基本语法

    bash


    /pr_comments [pr-number]

## 参数说明

```
引數| 描述
---|---
`[pr-number]`| 可選，指定要檢視的 PR 編號。如果不提供，則顯示當前分支的 PR

## 功能說明
執行 `/pr_comments` 命令後，Claude Code 會：

  1. 查詢相關的拉取請求
  2. 獲取 PR 的所有評論
  3. 顯示評論內容和作者
  4. 提供評論摘要

```yaml
## 使用场景

### 1\. 查看当前 PR 的评论

    bash


    # 查看当前分支的 PR 评论
    /pr_comments

    # 显示所有评论

### 2\. 查看特定 PR 的评论

    bash


    # 查看特定 PR 的评论
    /pr_comments 123

    # 显示 PR #123 的所有评论

### 3\. 了解审查反馈

    bash


    # 查看审查评论
    /pr_comments

    # 了解需要修改的地方

## 输出示例

    bash


    Pull Request #123 Comments
    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    - 2 个一般性评论

## 注意事项

```
  1. 需要配置 Git 整合
  2. 需要有訪問 PR 的許可權
  3. 評論會按時間順序顯示
  4. 可以根據評論進行程式碼修改

```
## 与其他命令的配合使用

    bash


    # 查看 PR 评论，然后修改代码
    /pr_comments
    # 根据评论修改代码

## 最佳实践

```
  1. **及時檢視** ：及時檢視 PR 評論，瞭解反饋
  2. **逐條處理** ：逐條處理評論，確保所有問題都得到解決
  3. **溝通確認** ：不確定的地方，及時與審查者溝通
  4. **持續改進** ：根據評論改進程式碼質量

## 總結
`/pr_comments` 命令是 Claude Code 中用於檢視 PR 評論的重要命令。透過使用此命令，你可以：

  * 檢視拉取請求的評論
  * 瞭解程式碼審查的反饋
  * 根據評論改進程式碼
  * 提高程式碼質量

合理使用 `/pr_comments` 命令可以幫助你更好地參與程式碼審查流程。

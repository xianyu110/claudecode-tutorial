# 2.2 VS Code插件安装

## 2.2.1 安装VS Code#

Claude Code提供了VS Code插件，因此在安装Claude Code插件之前，需要先确保已安装VS Code。

### 2.2.1.1 检查VS Code是否已安装#

打开终端或命令提示符，运行以下命令检查VS Code是否已安装：

    bash

    复制

    # 检查VS Code版本
    code --version

如果输出显示VS Code版本信息，则说明VS Code已安装，可以跳过安装步骤。

### 2.2.1.2 安装VS Code#

如果没有安装VS Code，可以通过以下方式安装：

#### 方式一：使用官方安装包

  1. 访问VS Code官方网站：<https://code.visualstudio.com/>
  2. 下载适合你操作系统的安装包
  3. 运行安装程序，按照提示完成安装

#### 方式二：使用包管理器（推荐）

**对于macOS：**

    bash

    复制

    # 使用Homebrew安装
    brew install --cask visual-studio-code

**对于Linux：**

    bash

    复制

    # Ubuntu/Debian
    wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
    sudo install -o root -g root -m 644 packages.microsoft.gpg /etc/apt/trusted.gpg.d/
    sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/trusted.gpg.d/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
    rm -f packages.microsoft.gpg
    sudo apt update
    sudo apt install code
    # Fedora
    sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
    sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'
    sudo dnf install code

## 2.2.2 安装Claude Code插件#

安装完VS Code后，现在可以安装Claude Code插件了。

### 2.2.2.1 从VS Code市场安装（推荐）#

  1. 打开VS Code
  2. 点击左侧边栏的**扩展** 图标（或使用快捷键`Ctrl+Shift+X`/`Cmd+Shift+X`）
  3. 在搜索框中输入`Claude Code`
  4. 在搜索结果中找到Claude Code插件，点击**安装** 按钮
  5. 安装完成后，插件会自动启用

### 2.2.2.2 手动安装插件#

如果无法从VS Code市场安装插件，可以使用以下方式手动安装：

  1. 访问Claude Code插件的GitHub仓库或官方网站，下载最新的VSIX文件
  2. 打开VS Code
  3. 点击左侧边栏的**扩展** 图标（或使用快捷键`Ctrl+Shift+X`/`Cmd+Shift+X`）
  4. 点击扩展面板右上角的**更多操作** 按钮（三个点）
  5. 选择**从VSIX安装**
  6. 选择下载的VSIX文件，点击**安装**
  7. 安装完成后，重新加载VS Code以启用插件

### 2.2.2.3 验证插件安装#

安装完成后，可以通过以下方式验证插件是否成功安装：

  1. 打开VS Code
  2. 点击左侧边栏的**Claude Code** 图标（如果已添加到边栏）
  3. 或者，使用快捷键`Ctrl+Shift+P`/`Cmd+Shift+P`打开命令面板
  4. 输入`Claude Code`，如果看到相关命令，则说明插件已成功安装

### 2.2.2.4 插件更新#

Claude Code插件会定期更新，提供新功能和修复bug。可以通过以下方式更新插件：

  1. 打开VS Code
  2. 点击左侧边栏的**扩展** 图标（或使用快捷键`Ctrl+Shift+X`/`Cmd+Shift+X`）
  3. 点击扩展面板右上角的**更多操作** 按钮（三个点）
  4. 选择**检查更新**
  5. 如果有更新可用，点击**更新** 按钮即可更新插件

或者，在扩展面板中找到Claude Code插件，如果有更新可用，会显示**更新** 按钮。

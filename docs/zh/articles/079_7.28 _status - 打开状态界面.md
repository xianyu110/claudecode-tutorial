# 7.28 /status - 打开状态界面

## 概述

`/status` 命令用于打开 Claude Code 的状态界面（状态选项卡），显示版本、模型、账户和连接性信息。

## 基本语法

    bash


    /status

## 功能说明


执行 `/status` 命令后，Claude Code 会：

  1. 打开状态界面
  2. 显示系统信息
  3. 显示模型信息
  4. 显示账户信息
  5. 显示连接状态

## 状态信息

### 系统信息


  * Claude Code 版本
  * 操作系统
  * Node.js 版本
  * 内存使用

### 模型信息

  * 当前使用的模型
  * 模型版本
  * 模型状态

### 账户信息

  * 登录状态
  * 账户类型
  * 使用限制

### 连接状态

  * API 连接状态
  * 网络延迟
  * 最后连接时间

```
## 使用场景

### 1\. 查看系统状态

    bash


    # 查看系统状态
    /status

    # 显示所有状态信息

### 2\. 检查连接

    bash


    # 检查 API 连接状态
    /status

    # 查看连接信息

### 3\. 查看使用情况

    bash


    # 查看使用情况
    /status

    # 显示使用限制和当前使用量

```

## 总结

`/status` 命令是 Claude Code 中用于查看状态的重要命令。通过使用此命令，你可以：

  * 查看系统信息
  * 查看模型信息
  * 查看账户信息
  * 查看连接状态

合理使用 `/status` 命令可以帮助你了解 Claude Code 的运行状态。

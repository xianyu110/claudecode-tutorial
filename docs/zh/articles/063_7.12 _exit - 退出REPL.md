# 7.12 /exit - 退出REPL

## 概述

`/exit` 命令用于退出 Claude Code 的交互式 REPL（Read-Eval-Print Loop）环境，结束当前会话。

## 基本语法

    bash


    /exit

## 功能说明


执行 `/exit` 命令后，Claude Code 会：

  1. 保存当前会话状态（如果配置了自动保存）
  2. 清理临时文件
  3. 关闭所有后台任务
  4. 退出 REPL 环境
  5. 返回命令行

```
## 使用场景

### 1\. 完成工作后退出

    bash


    # 完成所有工作后
    /exit

    # 退出 Claude Code

### 2\. 需要重启时退出

    bash


    # 需要重启 Claude Code 时
    /exit

    # 然后重新启动

### 3\. 切换项目时退出

    bash


    # 切换到另一个项目前
    /exit

    # 在新项目目录中重新启动

## 注意事项

```

  1. `/exit` 会立即退出，不会提示保存
  2. 未保存的对话历史会丢失
  3. 后台任务会被终止
  4. 配置和设置会保留

```bash
## 与其他命令的配合使用

    bash


    # 退出前清理
    /export session.md
    /exit

    # 执行流程：
    # 1. 导出当前对话到文件
    # 2. 退出 Claude Code

## 最佳实践

```

  1. **导出对话** ：退出前使用 `/export` 保存重要对话
  2. **清理后台** ：退出前使用 `/bashes` 检查后台任务
  3. **保存状态** ：确保重要工作已保存
  4. **正常退出** ：使用 `/exit` 而不是强制终止

## 总结

`/exit` 命令是 Claude Code 中用于退出 REPL 的命令。通过使用此命令，你可以：

  * 正常退出 Claude Code
  * 清理临时文件
  * 关闭后台任务
  * 保存会话状态

合理使用 `/exit` 命令可以帮助你安全地结束 Claude Code 会话。

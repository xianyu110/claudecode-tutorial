# 7.33 /vim - 进入vim模式

## 概述

`/vim` 命令用于进入 vim 模式，允许你使用 vim 的插入和命令模式交替输入文本。

## 基本语法

    bash


    /vim

## 功能说明


执行 `/vim` 命令后，Claude Code 会：

  1. 启用 vim 模式
  2. 切换到正常模式
  3. 允许使用 vim 快捷键
  4. 显示 vim 模式激活提示

## Vim 模式

### 正常模式 (Normal Mode)


  * **进入** : 默认模式
  * **功能** : 浏览和编辑文本
  * **快捷键** : h, j, k, l, dd, yy, p 等

### 插入模式 (Insert Mode)

  * **进入** : 按 `i` 或 `a`
  * **功能** : 输入文本
  * **退出** : 按 `Esc`

### 命令模式 (Command Mode)

  * **进入** : 按 `:` 或 `/`
  * **功能** : 执行命令
  * **退出** : 按 `Esc`

```
## 使用场景

### 1\. 进入 vim 模式

    bash


    # 进入 vim 模式
    /vim

    # 现在可以使用 vim 快捷键

### 2\. 编辑文本

    bash


    # 在 vim 模式下编辑文本
    /vim
    # 按 i 进入插入模式
    # 输入文本
    # 按 Esc 返回正常模式

### 3\. 浏览文本

    bash


    # 使用 vim 快捷键浏览文本
    /vim
    # 使用 h, j, k, l 移动光标

## 常用 Vim 快捷键

### 移动

```

  * `h`: 左移
  * `j`: 下移
  * `k`: 上移
  * `l`: 右移
  * `w`: 下一个词
  * `b`: 上一个词

### 编辑

  * `i`: 在光标前插入
  * `a`: 在光标后插入
  * `dd`: 删除行
  * `yy`: 复制行
  * `p`: 粘贴

### 退出

  * `Esc`: 返回正常模式
  * `:q`: 退出
  * `:w`: 保存
  * `:wq`: 保存并退出

## 注意事项

  1. Vim 模式会影响所有输入
  2. 需要了解 vim 基本操作
  3. 可以随时退出 vim 模式
  4. 不熟悉 vim 的用户可能需要学习


`/vim` 命令是 Claude Code 中用于启用 vim 模式的命令。通过使用此命令，你可以：

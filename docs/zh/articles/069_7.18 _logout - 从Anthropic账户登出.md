# 7.18 /logout - 从Anthropic账户登出

## 概述

`/logout` 命令用于从 Anthropic 账户登出，清除本地保存的访问令牌。

## 基本语法

    bash


    /logout

## 功能说明


执行 `/logout` 命令后，Claude Code 会：

  1. 清除本地保存的访问令牌
  2. 断开与 Anthropic API 的连接
  3. 显示登出成功确认
  4. 需要重新登录才能使用

```
## 使用场景

### 1\. 在公共设备上登出

    bash


    # 在公共设备上使用后
    /logout

    # 清除访问令牌，保护账户安全

### 2\. 切换账户前登出

    bash


    # 切换到另一个账户前
    /logout
    /login

    # 先登出，再登录新账户

### 3\. 令牌问题

    bash


    # 当令牌出现问题时
    /logout
    /login

    # 重新登录获取新令牌

## 注意事项

```

  1. 登出后无法使用 Claude Code
  2. 需要重新登录才能继续使用
  3. 本地保存的配置不会丢失
  4. 访问令牌会被完全清除
  5. 会话状态会被重置

## 与其他命令的配合使用

    bash


    # 登出后重新登录
    /logout
    /login

    # 执行流程：
    # 1. 登出当前账户
    # 2. 登录新账户或重新登录

## 最佳实践


  1. **公共设备** ：在公共设备上使用后，务必登出
  2. **账户切换** ：切换账户前先登出
  3. **定期登出** ：定期登出并重新登录，更新令牌
  4. **安全第一** ：不要在共享设备上保存令牌

## 总结

`/logout` 命令是 Claude Code 中用于账户安全的重要命令。通过使用此命令，你可以：

  * 从 Anthropic 账户登出
  * 清除本地访问令牌
  * 保护账户安全
  * 为切换账户做准备

合理使用 `/logout` 命令可以帮助你保护 Anthropic 账户的安全。

# 7.31 /todos - 列出当前待办事项

## 概述
`/todos` 命令用于列出当前待办事项，帮助你跟踪和管理任务进度。

```
## 基本语法

    bash


    /todos [subcommand]

## 子命令

```
子命令| 描述
---|---
无参数| 列出所有待办事项
`add`| 添加新的待办事项
`complete`| 标记待办事项为完成
`remove`| 删除待办事项
`clear`| 清除所有待办事项

```
## 功能说明

### 列出待办事项

    bash


    /todos

    # 显示所有待办事项及其状态

### 添加待办事项

    bash


    /todos add

    # 按照提示输入待办事项

### 完成待办事项

    bash


    /todos complete 1

    # 标记待办事项 #1 为完成

### 删除待办事项

    bash


    /todos remove 2

    # 删除待办事项 #2

## 使用场景

### 1\. 查看待办事项

    bash


    # 查看所有待办事项
    /todos

    # 了解当前任务列表

### 2\. 添加任务

    bash


    # 添加新的待办事项
    /todos add

    # 输入任务描述

### 3\. 完成任务

    bash


    # 标记任务为完成
    /todos complete 1

    # 更新任务状态

### 4\. 清理任务

    bash


    # 清除已完成的任务
    /todos clear

    # 清空待办事项列表

    bash


    待办事项列表

## 注意事项

```
  1. 待办事项保存在本地
  2. 待办事项不会自动同步
  3. 可以手动编辑待办事项文件
  4. 清除操作无法撤销

```
## 与其他命令的配合使用

    bash


    # 查看待办事项，然后完成任务
    /todos
    /todos complete 1

    # 执行流程：
    # 1. 查看待办事项列表
    # 2. 完成指定任务

## 最佳实践

```
  1. **定期更新** ：定期更新待办事项列表
  2. **明确任务** ：使用清晰的任务描述
  3. **优先级排序** ：按优先级排序任务
  4. **及时完成** ：及时完成任务并更新状态

## 总结
`/todos` 命令是 Claude Code 中用于任务管理的重要命令。通过使用此命令，你可以：

  * 查看待办事项列表
  * 添加新的任务
  * 标记任务为完成
  * 删除任务

合理使用 `/todos` 命令可以帮助你更好地跟踪和管理任务进度。

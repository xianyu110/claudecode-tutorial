# 7.17 /login - 切换Anthropic账户

## 概述
`/login` 命令用于切换 Anthropic 账户，允许你在不同的 Anthropic 账户之间切换。

```
## 基本语法

    bash


    /login

## 功能说明

```
执行 `/login` 命令后，Claude Code 会：

  1. 打开浏览器进行身份验证
  2. 提示你登录 Anthropic 账户
  3. 获取访问令牌
  4. 保存令牌到本地配置
  5. 切换到新账户

```
## 使用场景

### 1\. 切换账户

    bash


    # 切换到另一个 Anthropic 账户
    /login

    # 按照提示完成身份验证

### 2\. 重新认证

    bash


    # 当令牌过期时
    /login

    # 重新获取访问令牌

### 3\. 首次登录

    bash


    # 首次使用 Claude Code
    /login

    # 登录 Anthropic 账户

## 登录流程

```
  1. 执行 `/login` 命令
  2. 浏览器自动打开 Anthropic 登录页面
  3. 输入账户凭据
  4. 授权 Claude Code 访问
  5. 返回命令行，登录成功

## 注意事项
  1. 需要有效的 Anthropic 账户
  2. 需要互联网连接
  3. 令牌会保存在本地配置文件中
  4. 切换账户会影响当前会话
  5. 旧账户的令牌会被替换

```
## 与其他命令的配合使用

    bash


    # 登录后检查状态
    /login
    /status

    # 执行流程：
    # 1. 登录 Anthropic 账户
    # 2. 查看账户状态

## 最佳实践

```
  1. **账户安全** ：不要在公共设备上保存令牌
  2. **定期更新** ：定期重新登录以更新令牌
  3. **账户管理** ：为不同用途使用不同的账户
  4. **退出登录** ：在公共设备上使用后，记得 `/logout`

## 总结
`/login` 命令是 Claude Code 中用于账户管理的重要命令。通过使用此命令，你可以：

  * 切换 Anthropic 账户
  * 重新获取访问令牌
  * 管理多个账户
  * 保持账户安全

合理使用 `/login` 命令可以帮助你灵活地管理 Anthropic 账户。

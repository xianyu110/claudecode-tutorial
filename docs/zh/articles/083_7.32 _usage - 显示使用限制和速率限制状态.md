# 7.32 /usage - 显示使用限制和速率限制状态

## 概述
`/usage` 命令用于显示计划使用限制和速率限制状态，帮助你了解当前使用情况和剩余配额。

```
## 基本语法

    bash


    /usage

## 功能说明

```
执行 `/usage` 命令后，Claude Code 会：

  1. 显示当前使用量
  2. 显示使用限制
  3. 显示速率限制状态
  4. 显示配额剩余情况

```
## 使用信息

### 令牌使用

```
  * 当前使用量
  * 每日限制
  * 每月限制
  * 使用百分比

### 速率限制
  * 请求速率
  * 剩余请求数
  * 重置时间
  * 限制状态

### 计划信息
  * 当前计划类型
  * 计划限制
  * 升级选项

```yaml

## 计划类型

### 免费计划

```
  * 每日限制: 200,000 令牌
  * 每月限制: 1,000,000 令牌
  * 速率限制: 20 请求/分钟

### Pro 计划
  * 每日限制: 500,000 令牌
  * 每月限制: 5,000,000 令牌
  * 速率限制: 50 请求/分钟

### Enterprise 计划
  * 每日限制: 2,000,000 令牌
  * 每月限制: 20,000,000 令牌
  * 速率限制: 100 请求/分钟

## 注意事项
  1. 使用限制会在每日/每月重置
  2. 速率限制按分钟计算
  3. 超出限制后需要等待或升级计划
  4. 使用量实时更新

```
## 与其他命令的配合使用

    bash


    # 查看使用情况，然后优化

    # 执行流程：
    # 1. 查看使用情况
    # 2. 压缩对话以减少令牌使用

## 最佳实践

```
  1. **定期检查** ：定期查看使用情况
  2. **提前规划** ：提前规划使用量
  3. **优化使用** ：优化对话以减少令牌消耗
  4. **及时升级** ：接近限制时考虑升级计划

## 总结
`/usage` 命令是 Claude Code 中用于查看使用情况的重要命令。通过使用此命令，你可以：

  * 查看当前使用量
  * 查看使用限制
  * 查看速率限制状态
  * 了解剩余配额

合理使用 `/usage` 命令可以帮助你更好地管理和规划使用配额。
